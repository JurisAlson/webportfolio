EcoScrap Platform

Project Overview

EcoScrap is a Android mobile application developed to modernize the recycling workflow between residents, waste collectors, and junkshops. The project addresses the inefficiencies of manual recycling operations by providing a centralized platform for scheduling recyclable pickups, recording transactions, managing inventory, and improving communication among stakeholders. Built in collaboration with Mores Scrap Trading in Barangay Palo Alto, Calamba City, the application digitizes the junkshop's daily operations while encouraging community participation in plastic recycling. The platform also integrates AI-powered plastic classification using TensorFlow Lite to help residents identify recyclable plastics before submitting them for collection or drop-off. The project combines mobile development, cloud technologies, geolocation services, real-time communication, and machine learning into a single solution designed for small-scale recycling businesses.

Key Contributions

  • Foundational Development: Architected the core backbone, state management, and Firebase infrastructure.
  • Residence Module: Led end-to-end development of the Residence interface and scheduling logic.
  • Admin & RBAC System: Sole developer of the Admin Console and the Role-Based Access Control system for multi-tier user governance.
  • Junkshop & Collector Logic: Developed inventory tracking and route management engines.
  • Computer Vision: Engineered the ML pipeline using TensorFlow Lite for on-device waste classification.
  • Data Security: Implemented end-to-end data lifecycle security, including AES-256 encryption.
